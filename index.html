Description du Projet

Je travaille avec Dr Samer Nashed, un post-doctorant dans le laboratoire REAL de mon professeur Liam Paull. L'objectif de notre projet est de développer un modèle capable de prédire la performance des algorithmes SLAM (Simultaneous Localization and Mapping).

Les solutions actuelles au problème SLAM sont souvent spécifiques à des environnements particuliers. Or, pour résoudre réellement le problème SLAM de manière générale, il est essentiel de pouvoir fonctionner dans n'importe quel environnement. Plutôt que de créer un algorithme universel, une approche prometteuse, proposée par Dr Samer Nashed, est de développer un modèle capable d'évaluer la performance des différents algorithmes SLAM existants, chacun étant optimisé pour son environnement respectif.

L'idée est qu'avec un tel modèle, nous pourrions utiliser un système de planification pour gérer un portefeuille d'algorithmes SLAM, sélectionnant le plus performant en fonction de l'environnement donné. Notre projet vise donc à développer ce modèle prédictif en utilisant des réseaux de neurones convolutifs (CNN) et le dataset KITTI, composé d'images. Idéalement, ce modèle doit pouvoir prédire la performance des algorithmes SLAM pour une séquence d'images donnée.

Pour cela, nous allons utiliser des algorithmes SLAM open source, l'un basé sur des capteurs visuels (ORB-SLAM2) et l'autre sur des capteurs LIDAR (LOAM). Une fois les données collectées, nous passerons à la phase de développement du modèle en utilisant PyTorch. Cette phase nécessitera une recherche approfondie pour déterminer comment rendre le modèle suffisamment performant. Nous explorerons différentes techniques d'apprentissage machine et d'apprentissage profond, tout en évaluant si nos données sont suffisamment riches pour permettre le développement d'un modèle efficace.

En résumé, notre projet vise à améliorer la robustesse et la flexibilité des systèmes SLAM en développant un modèle capable de prédire dynamiquement la performance des algorithmes SLAM en fonction de l'environnement, ce qui est crucial pour les applications de robotique autonome.
