About the Project

This project aims to develop a performance model for SLAM (Simultaneous Localization and Mapping) algorithms in autonomous robotics.

Project Objectives

In an autonomous robotics system, it is crucial to operate in complex and dynamic environments. An autonomous robot must be capable of making critical decisions in difficult and ambiguous situations.

Under the supervision of Dr. Samer Nashed and Dr. Liam Paull at the REAL laboratory, this project aims to develop a performance model for SLAM algorithms. The goal is to enable the robot to better assess its perception and decision-making capabilities based on the environment.
